Git commands

Commands Purpose
$git init To create a new git repository
$git add "your files" Adds a change in the working directory to the staging area
$git commit -m "your message" Used to fetch updates from the staging area to the repository
$git remote add origin "Your repository url" Used to add a new remote repository to your local project
$git push -u origin master Used to upload local repository content to a remote repository
$git clone "Your repository url" Creates a local copy of a Git repository from a remote source
$git config --list To set git configuration values on a global or local project level
$git log Lists the commits made in that repository in reverse chronological order(arranging from newest to oldest)
$git config --global user.name "Your name" To set your user name
$git config --global user.email "Your email" To set your user email
$git config --global user.name To ensure your user name
$git config --global user.email To ensure your user email
$git branch/$git status To display the state of the working directory and the staging area.
$git diff To show the differences between various states of a repository.
$ssh-keygen To create new authentication key pairs for SSH
